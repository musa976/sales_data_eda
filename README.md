# 📊 Sales Data Exploratory Data Analysis (EDA)

This project performs **Exploratory Data Analysis (EDA)** on a sales dataset using **Python** and **Jupyter Notebook**. The goal is to clean the data, analyze sales trends, and create visualizations that provide meaningful business insights.

---

## 📌 Project Overview

This notebook includes:

- Data loading and inspection
- Data cleaning and preprocessing
- Handling missing values
- Descriptive statistics
- Sales trend analysis
- Data visualization
- Business insights

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/musa976/sales_data_eda.git
```

### 2. Open the project

```bash
cd sales_data_eda
```

### 3. Install required libraries

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### 4. Launch Jupyter Notebook

```bash
jupyter notebook
```

Open:

```
sales_data.ipynb
```
